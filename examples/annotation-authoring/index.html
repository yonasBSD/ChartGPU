Annotations - Comprehensive Demo

Interactive annotation system with full editing capabilities. Create, edit, drag, and delete annotations with a rich configuration dialog. Demonstrates all annotation types: reference lines (horizontal/vertical), point markers, text annotations (plot-space + data-space tracking), labels, and styling options.

✨ Try the new features:
Right-click empty space → Add vertical/horizontal line or text note with custom color, style & label
Click & drag annotations → Reposition them (lines constrained to their axis)
Right-click on annotation → Edit properties or delete
Use Undo/Redo buttons → All operations are reversible
